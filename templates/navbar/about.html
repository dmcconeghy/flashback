ABOUT FLASHBACK

I'm Dave McConeghy and Flashback is a capstone project I completed for my full-stack software engineering bootcamp. The site's front-end is styled with Bootstrap, and the project's back-end uses Python, Flask, and SQLAlchemy. I have avoided using Flask plugins such as Flask-Login or Flask-Restless in favor of practicing writing those API and CRUD features myself.

Perhaps you'll appreciate this project if you've ever had a moment where you heard a song and thought, "I remember when I first heard that on MTV Unplugged!" That's what happened to me when I happened to hear "All Apologies" a while back. I thought, "I remember when that came out. It was the fall of 1993 and Cobain killed himself the next Spring." It was such a strong memory about that tragedy, but what else I was listening to then?

A quick Google later and I found half a dozen albums that I had on repeat from that same moment. Off I went to re-live my musical fall of 1993 with Counting Crows, Pearl Jam, and Billy Joel. None of Nirvana's songs ever made the Billboard Hot 100 (though their MTV Unplugged album was by most measures the most popular album of 1994), but I hope this project offers users a similar opportunity by providing a simple tool to explore Billboard charts.

Data for Billboard charts and songs is not fetched prior to deployment, but rather collected dynamically based on user requests. This fulfulls project specifications for the use of external APIs, and it means that every user search helps build the site's data-scope. Explore all you like and pay-it-forward for future users by fetching more songs and image urls! Please note that I claim no rights for either Billboard chart data or images. This project remains a personal, educational exercise alone.

One limitation of this project's data is that chart results are exclusively for the Billboard's "Hot 100" (8/4/1958 - Present). Just as it would be nice to browser other charts like the Global 200, there are many other opportunities for improvements such as:

  • Integrating lyrics or other track/album information from Spotify/iTunes/Wikipedia/Genius
  • Embedded listening options or playlist export through Spotify authentication
  • Increasing the site's use of internal JSON API to enable loading-free features
  • Adding a search feature to find specific artists or songs
