ABOUT FLASHBACK

I'm Dave McConeghy and Flashback is a musical memory project I completed for my full-stack software engineering bootcamp. You'll appreciate this project if you've ever had a moment where you heard a song and thought, "I remember when I first heard that!"

That's what happened to me when I happened to hear Nirvana's "All Apologies" a while back. I thought, "I remember when that came out. It was 1993 and Cobain killed himself just a few months later." It was such a strong memory, but I couldn't think of what else I was listening to then.

A quick Google later and I found half a dozen albums that I was listening to on repeat from that same moment. Off I went to re-live my musical fall of 1993 with Counting Crows, Pearl Jam, and Billy Joel. I hope this project offers users a similar experience by providing a Billboard chart based timeline to explore.

The Front-End is styled with Bootstrap, and the project was built with HTML, CSS, Javascript, Python, Flask, and SQLAlchemy. I have avoided using Flask plugins such as Flask-Login or Flask-Restless in favor of practicing writing such features myself.

Data for Billboard charts and songs is not fetched ahead of time, but rather collected dynamically based on user requests. This fulfulls project specifications for the use of APIs, and it means that every user search helps build the site's data-scope. Explore all you like and pay-it-forward for future users!

One notable limitation of this project's data is that chart results are exclusively for the Billboard's "Hot 100" (8/4/1958 - Present). Among the future features this project could include are:

  • Offering results from other charts (such as the Global 200 or genre-charts like R&B/Hip-Hop or Country)
  • Adding additional song and track data from API integrations to Spotify/iTunes/Wikipedia
  • Embedded listening options or playlist creation features with Spotify authentication
  • Provide additional context for chart dates such as major world events, sports results, or names of elected officials
  • Relying on pre-fetched Billboard data and instead using an internal API for results, enabling far more dynamic search options.
